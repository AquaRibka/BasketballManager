# BasketballManager

Starter repository for the Basketball Manager project.

## Structure

- `apps/backend/` - backend application
- `apps/frontend/` - frontend application
- `packages/simulation-engine/` - basketball simulation logic
- `packages/shared/` - shared code and contracts
- `docs/` - project documentation
- `infra/` - local infrastructure configuration

## Getting Started

Project scaffolding is initialized and ready for the first implementation steps.

Local PostgreSQL setup is described in `docs/postgres-local.md`.
